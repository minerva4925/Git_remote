* pwd -- print working directory
*mkdir -- make directory
* cd -- change directory
* ls -- display list
* touch -- create new file
* rm -- delete remove file

-----------------------
* Repository -- stores file/folder
	> Local repository -- version of repo stored on pc
	> remote repository -- server cloud based

* git init -- initialize folder

* git config --global user.name
* git config --global user.email

* git status -- current status

* git add . --- adds all modified files in your current directory to the staging area

* git add "filename" --- add specific file to staging area

* git commit -m "message" --- saves changes (msg describes the changes)

* git log --- show all the commits that were made

*git log --oneline --- show condensed commits that were made
